Skytrax Airline Reviews Sentiment Analysis

Project Overview  
This project is a web-based sentiment analysis system developed using Python and Streamlit.  
The system analyzes airline customer reviews and classifies them into positive, negative, or neutral sentiments.  
It helps in understanding customer feedback and service quality.

Objectives  
- Analyze airline customer reviews  
- Perform sentiment classification using NLP  
- Visualize sentiment distribution  
- Predict sentiment for user-entered reviews  

Dataset Description  
The project uses four datasets:
airline.csv – Airline reviews  
airport.csv – Airport reviews  
lounge.csv – Lounge reviews  
seat.csv – Seat comfort reviews  

Each dataset contains customer review text and related information.

Technologies Used  
Python  
Streamlit  
Natural Language Processing (NLTK, TextBlob)  
Machine Learning (TF-IDF, Logistic Regression)  
Matplotlib and Seaborn  

Methodology  
1. Load and combine all datasets  
2. Clean and preprocess the text data  
3. Perform sentiment analysis  
4. Convert text data into numerical features using TF-IDF  
5. Train a Logistic Regression model  
6. Analyze and visualize sentiment results  
7. Predict sentiment for new reviews  

Application Features  
- Web-based interactive interface  
- Dataset overview and insights  
- Sentiment distribution analysis  
- Category-wise sentiment visualization  
- Real-time sentiment prediction  

Conclusion  
This project demonstrates the use of natural language processing and machine learning techniques to analyze airline customer reviews.  
It provides useful insights into customer opinions and overall service quality.
